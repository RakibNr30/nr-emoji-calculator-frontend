# Emoji Calculator Frontend

## Installation
Step-1: <br/>
Run the command to clone the repository:
```bash
git clone https://github.com/RakibNr30/nr-emoji-calculator-frontend.git
```
Step-2:<br/>
Checkout to ``development`` branch
 ```bash
 git checkout development
 ```
Step-3:<br/>
Install the dependencies
 ```bash
npm install
```
Step-4:<br/>
Setup and run laravel backend from (if not run yet)
```bash
https://github.com/RakibNr30/nr-emoji-calculator-backend.git
```
Step-5:<br/>
Start the application by running the following command
 ```bash
npm run dev
```
Step-6:<br/>
You can access the application from
 ```bash
http://localhost:3000
```
